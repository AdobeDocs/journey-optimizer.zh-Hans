# AI支持的品牌校正 {#brand-corrections}

>[!AVAILABILITY]
>
>此功能适用于Adobe Journey Optimizer，并适用于推送、短信和Web内容渠道。

## 概述 {#overview}

在品牌QA期间标记内容时，Adobe Journey Optimizer可以使用创作AI自动生成更正或改进的替换文本。 您不会手动重写标记的副本，而是会收到与您的品牌指南相一致的内联建议 — 只需单击一下即可查看、预览和应用这些建议。

这会将Brand QA步骤从阻止审核审核审核审核门转为&#x200B;**即用即用体验**，从而减少手动更正所花费的时间并加快跨渠道的内容生产。

此功能专为需要保持大容量、多渠道营销活动中的品牌合规性，同时又不降低生产工作流程的内容营销人员、撰稿人和营销活动操作员而设计。

## 工作原理 {#how-it-works}

Adobe Journey Optimizer中的Brand QA根据贵组织的品牌指南（包括语调、术语、消息传递标准和编辑规则）来评估您的内容。 当检测到违规或质量问题时，系统会标记受影响的内容元素，并在支持的情况下，使用Adobe创作AI引擎自动生成推荐的替换。

端到端流程如下所示：

1. **品牌QA扫描** — 当您对内容（推送通知正文、SMS消息文本或Web内容块）运行品牌检查时，系统会根据有效的品牌规则评估每个元素。
2. **违规检测** — 未能通过一个或多个品牌标准的元素将标有严重性指示器（例如，严重、警告或建议）。
3. **AI建议生成** — 对于每个标记的元素，系统会自动生成一个或多个已更正的替换文本。 建议由Adobe创作AI引擎生成，该引擎知道违规原因和您的品牌指南上下文。
4. **内联预览** — 建议的替换文本将内联显示，直接与标记的原始文本旁边。 在进行任何更改之前，您可以并排比较原始版本和建议版本。
5. **一键应用** — 如果建议符合您的需要，请选择&#x200B;**应用**&#x200B;以将标记的文本替换为AI生成的版本。 内容编辑器将立即更新，并且品牌标志将被清除。
6. **手动覆盖** — 您从未被锁定在AI建议中。 您可以在应用建议之前对其进行编辑、将其关闭并自行编写更正，或者在上下文允许异常时完全忽略标记。

>[!NOTE]
>
>AI生成的建议仅为建议。 您的团队保留完全的编辑控制权。 在应用建议之前，请始终在营销策划的上下文中查看建议。

## 先决条件 {#prerequisites}

在使用AI支持的品牌校正之前，请确保满足以下条件：

- **已配置品牌准则** — 您的组织必须在Adobe Journey Optimizer中至少设置一个有效的品牌配置文件。 品牌配置文件定义了内容评估所依据的规则。 请联系您的Adobe管理员或品牌管理员，以验证是否已发布品牌配置文件并将其与您的沙盒关联。
- **已启用Content AI功能** — 必须为您的组织和沙盒启用AI支持的内容协助。 这在Adobe Admin Console中的产品配置文件级别进行管理。 如果在品牌扫描后未显示AI建议，请与管理员确认AI内容生成权利是否处于活动状态。
- **支持的内容类型** — 以下内容类型支持带有AI建议的品牌更正：推送通知标题和正文、短信消息文本以及通过Journey Optimizer Web Designer编辑的Web内容块。 富媒体资产（图像、视频）将单独评估品牌合规性，并且不在文本级AI更正的范围中。
- **编辑权限** — 您必须对标记的内容所在的历程、营销活动或内容模板具有编辑访问权限。

## 配置 {#configure}

如果您的组织已使用Brand QA，则无需额外配置即可启用AI支持的品牌校正。 在支持的内容类型中检测到品牌违规时，AI建议层会自动激活。

要运行品牌检查并访问AI建议：

1. 在Adobe Journey Optimizer中打开您的营销活动或旅程。
2. 导航到相关渠道（推送、短信或Web）的内容步骤。
3. 选择&#x200B;**检查品牌对齐方式**（或从内容编辑器工具栏中打开&#x200B;**品牌QA**&#x200B;面板）。
4. 等待扫描完成。 标记的元素在画布中高亮显示，并在右侧的&#x200B;**品牌问题**&#x200B;面板中列出。
5. 对于每个标记的项目，展开问题行以查看违规原因和AI生成的建议。
6. 使用&#x200B;**预览**&#x200B;查看内容画布中呈现的建议文本。
7. 选择&#x200B;**应用建议**&#x200B;以替换标记的文本，或选择&#x200B;**编辑**&#x200B;以在应用之前修改建议。
8. 解决或确认所有问题后，重新运行品牌检查以确认合规性。

>[!NOTE]
>
>应用建议会更新编辑器中的实时内容，但不会自动发布或激活营销策划。 您必须正常完成剩余的Campaign配置和激活步骤。

### 使用多个建议 {#multiple-suggestions}

对于某些违规，AI引擎可能会生成多个替代项。 在这些情况下，**品牌问题**&#x200B;面板会显示选项编号列表。 使用前进和倒退箭头可在备选方案之间循环，然后再选择最适合您意图的备选方案。 每个替代品都是用不同的文体方法生成的 — 例如，一个可以优先选择简洁，而另一个强调语调对齐。

### 批量更正 {#bulk-corrections}

如果同一内容片段中的多个元素已标记，您可以单独应用建议，或使用&#x200B;**品牌问题**&#x200B;面板顶部的&#x200B;**应用所有建议**&#x200B;操作在单个步骤中接受所有AI生成的替换。 在使用批量应用之前，请仔细查看列表，因为此操作会同时替换所有标记的文本。

>[!NOTE]
>
>仅当所有标记的项目都具有关联的AI建议时，批量应用才可用。 已跳过没有可用建议的项目（例如，图像合规性标记），仍将其标记以供手动审查。
