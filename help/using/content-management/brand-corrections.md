# AI支持的品牌校正和编辑建议 {#brand-corrections-ai}

>[!AVAILABILITY]
>
>此功能对具有有效Brand Guidelines和AI Assistant权利的Adobe Journey Optimizer用户可用。 有关访问详细信息，请联系您的Adobe代表。

## 概述 {#overview}

Adobe Journey Optimizer扩展了其GenAI支持的品牌合规工具，使其涵盖所有受支持的内容渠道 — 短信、推送通知、Web内容和电子邮件。 当品牌QA引擎检测到内容违反品牌准则或未通过编辑质量标准时，系统会自动生成更正的替代项，您可以直接在创作工作流中查看和应用。

这会将品牌验证从一个被动的核对清单转变为主动、随用随修的体验。 内容作者不会识别违规并返回编辑器手动解决每个问题，而是会收到人工智能生成的针对性建议，这些建议符合组织既定的品牌声音、语调和风格指南 — 所有这些都无需离开内容编辑器。

此功能专为需要快速将内容从草稿移至激活状态，同时保持大规模品牌合规性的内容营销人员、撰稿人和营销活动操作员而设计。

## 先决条件 {#prerequisites}

在使用AI支持的品牌校正之前，请确认以下内容：

- 在Adobe Journey Optimizer中配置了&#x200B;**品牌指南**。 如果没有有效的品牌配置文件，系统无法生成品牌一致的更正建议。
- 已为您的Adobe Experience Cloud组织启用&#x200B;**AI助手**&#x200B;功能。
- 您具有创建和编辑相关渠道（短信、推送、Web或电子邮件）内容的相应权限。
- 在Journey Optimizer内容编辑器中至少有一个内容项可供查看。

>[!NOTE]
>
>品牌更正建议是使用Adobe的创作AI基础架构生成的，并受适用于您组织的AI助手使用政策的约束。 发布之前请务必查看已接受的建议。

## 工作原理 {#how-it-works}

品牌校正直接集成到Journey Optimizer内容编辑器中的品牌QA验证流中。 端到端流程的工作方式如下。

**步骤1 — Brand QA扫描**：当您对内容运行品牌验证检查时（手动从审核面板运行或由工作流规则触发），系统会根据您配置的品牌指南评估每个内容块。 检查涵盖语调、术语、禁止使用的语言、编辑标准和监管要求。

**步骤2 — 违规检测和标记**：任何不符合品牌或编辑质量标准的内容区段都将使用违规指示器标记。 违规类型（例如，音调不匹配、禁止使用术语或不符合准则）与标记的区段一起显示，以便作者确切了解需要更改的内容。

**步骤3 — AI建议生成**：对于每个标记的区段，Journey Optimizer会使用AI助手自动生成一个或多个更正的替代项。 建议以您积极的品牌准则为基础，确保推荐的文本反映特定于贵组织的正确声音、术语和编辑风格。

**步骤4 — 内联预览和审核**：建议的更正将内联显示，直接与“品牌QA”侧面板中的标记内容相邻。 您可以比较原始文本与AI生成的替代文本，而无需导航离开编辑器。

**步骤5 — 接受或取消**：通过单击接受建议以使用更正后的版本替换标记的内容。 或者，驳回建议并手动编辑内容。 接受建议会立即更新内容块，并在QA面板中将违规标记为已解决。

**步骤6 — 重新验证**：应用更正后，请重新运行Brand QA扫描，以确认在发布或激活历程或营销活动中的内容之前已解决所有违规。

## 配置 {#configure}

除标准先决条件外，无需其他配置。 当品牌配置文件与您的内容关联并且已为您的组织启用AI助手时，该功能会在Brand QA面板中自动激活。

要开始使用AI支持的品牌校正，请执行以下操作：

1. 打开相关渠道的内容编辑器 — 短信、推送通知、Web或电子邮件。
2. 在编辑器工具栏中，选择&#x200B;**品牌指南**，然后从下拉菜单中选择适用的品牌配置文件。
3. 草稿或打开您的内容，然后从审核面板中选择&#x200B;**品牌QA**&#x200B;以启动扫描。
4. 查看&#x200B;**品牌QA**&#x200B;侧面板中标记的违规。 对于每个标记的项目，将自动显示一个AI生成的建议。
5. 单击&#x200B;**应用**&#x200B;接受建议，或单击&#x200B;**消除**&#x200B;手动处理更正。
6. 重新运行&#x200B;**品牌QA**&#x200B;以验证所有违规均已解决，然后继续执行标准审批或激活工作流。

### 支持的渠道 {#supported-channels}

在Adobe Journey Optimizer中的以下内容类型中提供了AI支持的品牌校正：

| 渠道 | 支持的内容元素 |
|---|---|
| **电子邮件** | 主题行、前置词、正文文本、CTA标签 |
| **短信** | 消息正文 |
| **推送通知** | 标题，正文文本 |
| **Web** | 标题、正文、按钮标签 |

>[!NOTE]
>
>支持的内容元素可能因您的渠道配置和品牌配置文件中定义的品牌指南而异。 图像和可视资产验证不属于AI生成的文本更正的范围。

## 主要优点 {#key-benefits}

**减少手动编辑工作量**：内容作者不再需要手动交叉引用每个标记问题的品牌指南。 AI建议会呈现现成的替代方案，从而显着缩短在修正周期中花费的时间。

**一致的品牌合规性**：更正基于用于验证的相同品牌准则。 接受的建议会保持与所有渠道中批准的品牌声音和编辑标准的一致性，从而减少多渠道营销活动中出现消息传送不一致的风险。

**加快内容生产**：通过将品牌QA转换为简化的随用即修工作流程，团队可以更快地移动内容审核周期，从而缩短草稿和激活之间的周转时间。 Campaign操作员只需一次即可解决整组违规，而无需在工具之间切换。

**跨渠道覆盖**：无论是生成短信促销活动、移动推送序列还是Web应用程序内消息，品牌校正建议在所有受支持的内容表面中都始终可用，从而确保品牌标准在您的品牌通信的任何地方都得到维护。
