# AI助手提示最佳实践 {#ai-assistant-prompting-guide}

>[!CONTEXTUALHELP]
>id="ajo_ai_assistant_prompt"
>title="提示示例"
>abstract="浏览Journey Optimizer文档，了解如何创建有效的提示，以生成高转化率的品牌内营销内容。"

本指南可帮助您构建请求、清晰地传达意图，并确保AI生成的消息符合您的品牌准则、受众需求和营销活动目标。
了解如何编写有效的提示，以使AI助手能够根据您的目标生成高质量、品牌化的营销内容。

## 使用CO-STAR框架 {#costar-framework}

要使AI助手达到最佳效果，请使用CO-STAR框架组织提示。 这种结构化方法可确保AI准确了解您所需的内容。

| 组件 | 它的含义 | 为什么这很重要 |
|-|-|-|
| **C — 上下文** | 有关营销活动、产品或情况的背景 | 帮助AI了解整体情况 |
| **O — 目标** | 您的特定营销目标 | 推动内容应实现的目标 |
| **S — 样式** | 您希望如何通信 | 设置方法 |
| **T — 音调** | 情感风格和声音 | 塑造您的信息感受 |
| **A — 受众** | 您定位的受众 | 确保消息与合适的人员引起共鸣 |
| **R — 要求** | 特定限制或必备项 | 定义边界和关键元素 |

### 质量核对清单 {#quality-checklist}

在生成内容之前，请确保满足以下条件：

&amp;amp；check； **清除目标**：清楚地声明操作、产品/服务、值和上下文。

&amp;amp；check； **已定义目标受众**：指定人口统计、角色或区段。

&amp;amp；检查； **内容类型对齐方式**：目标与所选渠道或格式匹配。

&amp;amp；check； **已配置的下拉选项**：选择了音频、策略和区域设置，请勿将其包含在提示中。

&amp;amp；check； **指定的文档焦点**：突出显示要引用的内容或部分。

&amp;amp；check； **已应用品牌**：已选择适当的品牌准则。

&amp;amp；check； **实际范围**：避免请求布局更改、样式或结构编辑。

## 编写有效的营销目标 {#marketing-objectives}

### 具体和注重行动

在制定营销目标时，请确保这些目标明确、可操作且可衡量。 避免使用Vague或泛型语句。

**良好目标示例：**

&amp;amp；check；“推动注册以免费试用新的AI支持的分析仪表板，为期30天”

&amp;amp；check；“为将于3月15日举行的‘将云成本降低40%’的B2B网络研讨会创造商机”

&amp;amp；check；“促销高级订阅的限时25%假日折扣，截止到12月25日”

**要避免的示例：**

✕ “促销我们的产品”（太模糊）

✕ “让人们购买东西”（价值不明）

✕ “关于新功能的电子邮件”（目的不足）

### 构建目标

始终提供上下文和价值主张，以便AI可生成相关内容。
使用此公式可以帮助您编写有效的目标：**操作+产品/服务+值/收益+紧急情况/上下文**

**良好目标示例：**

&amp;amp；check；“鼓励下载我们的新移动应用程序，帮助用户通过个性化的环保推荐跟踪可持续生活习惯”

&amp;amp；check；“为营销专业人员推广我们关于高级数据可视化技术的独家研讨会的注册服务”

&amp;amp；check；“提高产品发布活动的出席率，展示革命性的AI编写助手，每周可节省5小时以上的时间”

**要避免的示例：**

✕“宣布新应用程序”（缺少值主张和上下文）

✕ “让人员注册参加研讨会”（缺少有关受众和优势的具体说明）

✕“提升事件”（无明确的操作、值或紧迫性）

## 创建新内容与修改现有内容 {#new-vs-modify}

明确指示您的请求是否涉及生成新内容或更新现有材料。 这种区分很重要，因为它指导人工智能选择适当的方法，并确保产生更准确和有益的结果。

### 创建新内容

在启动营销活动、发布新产品或启动任何类型的更新或刷新通信时应用此策略。 它确保您的消息开始强有力并与目标保持一致。

**如何提示** ➤创建新内容时，请专注于营销目标而不引用现有内容。

>[!BEGINSHADEBOX]

**示例：**

* **SaaS试用版**：“向小型企业所有者推出我们的新CRM软件，节省时间达50%，潜在客户认证速度加快90%，并提供包含个性化入门的30天免费试用版”
* **功能发布**：“宣布新的API集成功能，将企业客户的开发时间减少60%，面向技术决策者”
* **活动促销活动**：“推动我们关于‘2024年数字营销趋势’的网络研讨会的注册，注册者将来自Google、Meta和Adobe的专家，强调可操作的见解和有限的名额（最多500个）”

>[!ENDSHADEBOX]

### 修改现有内容

>[!TIP]
>
>对于精细、汇总或简化等标准修改，请使用&#x200B;**优化**&#x200B;功能而不是编写自定义提示。 [了解详情](generative-uc.md##refine)

在需要更新、刷新或调整当前营销活动时应用此项。 此方法支持增量改进，确保消息传送保持相关状态而不从头开始。

**如何提示** ➤修改现有内容时，请明确指定要更改的内容以及如何更改内容。

>[!BEGINSHADEBOX]

**示例：**

* **Campaign更新**：“修改产品发布电子邮件，将重点放在企业安全功能上，而不是一般的生产力优势上，并将目标定位为具有合规性认证的IT决策者”
* **受众透视**：“调整我们的软件演示邀请，使其专门面向医疗保健，将通用示例替换为医疗保健用例和HIPAA合规性好处”
* **季节性更新**：“更新我们针对第4季度假期购物开展的第三季度促销活动，将重点从返校转为强调快速送货的假日礼品馈送”

>[!ENDSHADEBOX]

## 使用高级设置增强提示 {#personalize-prompt}

### 沟通策略类型

>[!TIP]
>
>使用文本设置菜单中的通信策略下拉列表，而不是在提示中提及它以进行专门处理。

您的沟通策略决定了如何展示您的消息以最大限度地扩大影响。 不同的策略可以更好地用于不同的目标，无论您是要建立紧迫感、建立信任，还是要立即采取行动。 选择最符合您的营销活动目标和受众思维的策略。

| **策略** | **最适合** | **消息样式** | **示例** |
|--------------|--------------|------------------------|--------------|
| **紧急** | 对时间敏感的优惠、截止日期、立即行动 | 使用基于时间的语言产生即时压力 | “立即行动：优惠将在午夜到期”<br>“在广告点满之前立即注册”<br>“立即开始48小时闪购” |
| **FOMO（害怕丢失）** | 限量优惠、事件、独占访问 | 使用紧迫性、稀缺性和时间压力提示 | “只剩下24小时了！ Limited stock at 40% off&quot; <br> &quot;最后机会：早盘定价明天结束&quot; <br> &quot;仅剩下100个Beta测试版点&quot; |
| **社交校对** | 建立信任、口碑、受欢迎的产品 | 利用他人的经验和验证 | “加入50,000多个满意的客户”<br>“行业专业人员评为4.9/5星”<br>“受财富500强公司信任” |
| **稀缺** | 库存有限、独占发放、高需求物料 | 强调有限的可用性和排他性 | “库存只剩下5件”<br>“限量版：100件可用”<br>“独家发行：先到先得” |
| **奖励** | 促销、奖励计划、特殊优惠 | 突出显示的实实在在的好处和回报 | “第一张订单享受20%的优惠”<br>“本周末可获得双点优惠”<br>“超过50美元的订单免运费” |
| **排他性** | Premium产品、VIP体验、仅限成员的选件 | 使用高级定位和特殊访问语言 | “独家邀请参加我们的私人预览”<br>“高级会员资格解锁高级分析”<br>“加入精选《财富》500强公司中已有我们使用过的公司” |
| **Gamification** | 参与活动、忠诚度计划、交互式内容 | 使用游戏机制和成就语言 | “解锁您的下一个奖励级别”<br>“完成挑战以获得徽章”<br>“获得独家奖项的排行榜” |
| **信息** | 教育、研究、复杂产品、思想领导力 | 使用事实、数据、见解和解释 | “分析10,000多种交互获得的5个见解” <br> “新研究揭示了3个突破性方法” <br> “在营销中实施人工智能的完整指南” |
| **教育和见解** | 学习内容、行业趋势、最佳实践 | 提供宝贵的知识和可操作的外包 | “使用我们的专家指南掌握高级技术”<br>“发现7个顶级营销人员使用的策略”<br>“了解如何通过5个步骤优化您的工作流” |

### 设定正确的语调 {#tone}

>[!TIP]
>
>使用“文本设置”菜单中的“色调”下拉菜单，而不是在提示中指定。

音调决定受众如何感知和响应您的消息。 始终选择与您的品牌声音和个人资料历程阶段一致的语调。

使用下表详细探究每种色调，包括何时效果最佳以及适合每种样式的内容示例。

| 色调 | 最适合 | 内容示例 |
|-|-|-|
| 专业 | B2B通信、正式公告 | “我们很高兴地宣布我们的战略伙伴关系……” |
| 同理心 | 客户支持，敏感主题 | “我们明白这对你来说一定是多么令人沮丧……” |
| 幽默 | 参与营销活动，突出显示内容 | “警告：可能会导致生产率大幅提高！” |
| 令人兴奋 | 产品发布、事件促销 | “这是你一直在等待的时刻！” |
| 励志 | 激励活动，品牌目的 | “我们携手合作，就能改变世界……” |
| 有说服力 | 促销活动、转化 | “不要错过这个有限的时间…… |
| 友好 | 客户参与，欢迎消息 | “我们很高兴你来到这里！” |
| 正式 | 法律通讯、正式通知 | “现将下列变更通知如下……” |
| 抱歉 | 服务恢复，问题解决 | “对于由此造成的不便，我们深表歉意……” |
| 自信 | 领导力内容，权威消息 | “这是你现在需要做的……” |
| storytelling | 品牌叙事，情感联系 | “这一切都始于一个简单的问题……” |
| 对话 | 培养活动，建立关系 | “让我们谈谈这能如何帮助你……” |

### 优化您的品牌资产 {#brand-assets}

>[!TIP]
>
>如果您已经通过&#x200B;**Brand Assets**&#x200B;菜单上传了品牌资产，则无需在提示中引用它。 系统会自动使用任何选定的文档。

品牌资产提供真实信息，通过特定、准确的详细信息丰富您的生成内容。
上传广泛文档（如产品小册子）时，在提示中添加要重点关注的部件：

* **不要使用** _“使用产品手册”_ **您应该使用** _“专注于高级安全功能和合规性认证，特别是SOC 2合规性和数据加密”_

* **而不是** _“参考案例研究”_ **您应该使用** _“突出医疗保健客户的ROI结果，特别是地区医疗中心的40%成本降低”_

* **您应该使用** _“强调API集成功能和开发人员优势，重点是REST API端点和99.9%的正常运行时间SLA”_，而不是&#x200B;****“包括技术详细信息”__

### 内容细化

>[!TIP]
>
>当您想要精细、汇总、简化、更改色调或翻译现有内容时，请使用优化功能而不是提示。 [了解详情](generative-uc.md#refine)

生成内容后，使用&#x200B;**优化**&#x200B;功能通过以下选项对其进行迭代和增强：

| **操作** | **用例** | **提示示例** |
|-|-|-|
| **精简** | 向简短内容添加深度和详细信息 | “阐述我们的安全功能的技术优势” |
| **摘要** | 压缩不同格式的冗长内容 | “为社交媒体帖子概述此产品” |
| **简化** | 使复杂的内容更易于访问 | “简化对一般受众的技术说明” |
| **更改音调** | 根据不同的受众调整内容 | “对年轻人来说，语气要变的更随意” |
| **Transcreate** | 翻译以外的文化适应 | “为日本市场打造这一营销活动” |
