# trim {#trim}

删除起始和结束空格。

## 类别

字符串

## 函数语法

`trim(<parameters>)`

## 参数

| 参数 | 类型 |
|-----------|------------------|
| 字符串 | 字符串 |

## 签名和返回的类型

`trim(<string>)`

返回字符串。

## 示例

`trim(" Hello ")`

返回“Hello”。
