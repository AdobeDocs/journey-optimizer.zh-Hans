# uuid {#uuid}

## 类别

字符串

## 函数语法

`uuid()`

## 参数

此函数不需要参数。

## 签名和返回的类型

`uuid()`

返回字符串。

## 示例

`uuid()`

返回“79e70b7f-8a85-400b-97a1-9f9826121553”。s
