# round {#round}

Returns the closest integer value to the argument with ties rounding to positive infinity.

## 类别

数学

## 函数语法

`round(<parameters>)`

## 参数

* 小数
* 整数

## 签名和返回类型

`round(<decimal>)`

`round(<integer>)`

返回整数。

## 示例

`round(3.14)`

返回3。

`round(3.54)`

返回4。

`round(-3.14)`

返回–3。

`round(3)`

返回3。
