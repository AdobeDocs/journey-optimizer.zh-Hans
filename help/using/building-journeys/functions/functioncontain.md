# contain {#contain}

检查第二个参数字符串是否包含在第一个参数字符串中。

## 类别

字符串

## 函数语法

`contain(<parameters>)`

## 参数

* 字符串

## 签名和返回的类型

`contain(<string>,<string>)`

返回布尔值。

## 示例

`contain("rowing is great", "great")`

返回true。
