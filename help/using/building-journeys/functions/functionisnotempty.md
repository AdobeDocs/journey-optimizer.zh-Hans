# isNotEmpty {#isNotEmpty}

如果参数中的字符串不为空，则返回true。

## 类别

字符串

## 函数语法

`isNotEmpty(<parameters>)`

## 参数

* 字符串

## 签名和返回的类型

`isNotEmpty(<string>)`

返回布尔值。

## 示例

`isNotEmpty("")`

返回false。

`isNotEmpty("hello")`

返回true。
