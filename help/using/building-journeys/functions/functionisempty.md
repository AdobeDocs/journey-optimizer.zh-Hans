# isEmpty {#isEmpty}

如果参数中的字符串不含字符，则返回true。

## 类别

字符串

## 函数语法

`isEmpty(<parameters>)`

## 参数

* 字符串

## 签名和返回的类型

`isEmpty(<string>)`

返回布尔值。

## 示例

`isEmpty("")`

返回真。

`isEmpty("Hello World")`

返回假。
