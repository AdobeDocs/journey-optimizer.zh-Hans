# currentTimeInMillis {#currentTimeInMillis}

返回新纪元毫秒的当前时间。

## 类别

日期

## 函数语法

`currentTimeInMillis()`

## 参数

此函数不使用参数。

## 签名和返回类型

`currentTimeInMillis()`

返回整数。

## 示例

`currentTimeInMillis()`

返回“1544712617131”。
