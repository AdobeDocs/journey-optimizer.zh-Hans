# indexOf {#indexOf}

檢查第一個引數字串是否以特定字串結尾（第二個引數字串）。 如果找不到字串，則傳回–1。

## 类别

字符串

## 函式語法

`indexOf(<parameters>)`

## 参数

| 参数 | 类型 |
|-----------|------------------|
| 字符串 | 字符串 |
| 指定的值 | 字符串 |

## 簽章和傳回的型別

`indexOf(<string>,<string>)`

傳回整數。

## 示例

`indexOf("Hello", "l")`

傳回2。

解释:

在「Hello」中，「l」的第一次出現位置為位置2。
