# lower {#lower}

返回参数的小写版本。

## 类别

字符串

## 函数语法

`lower(<parameter>)`

## 参数

* 字符串

## 签名和返回的类型

`lower(<string>)`

返回字符串。

## 示例

`lower("A")`

返回“a”。
