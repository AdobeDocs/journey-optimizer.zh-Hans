- **自定义操作**：调用外部API以向第三方系统&#x200B;*（特定于产品）发送数据或从第三方系统接收数据的历程活动*

- 每个用例示例包括应针对特定需求定制的建议；它们是起点，而不是规范性配置
- 数据集查询需要访问Adobe Experience Platform查询服务
