# 通过用例学习 {#jo-uc-gs}

本节将收集多个实际用例，以帮助您充分利用Adobe Journey Optimizer。 无论您是寻找战术模式（如抑制逻辑、个性化技术或历程退出策略），还是寻找涵盖营销和技术工作流的完整端到端场景，您都可以找到以下相关示例的链接。

在设计历程、建模数据或构建激活逻辑时，使用此库作为参考点。 每个示例都包含可针对特定需求定制的建议。
