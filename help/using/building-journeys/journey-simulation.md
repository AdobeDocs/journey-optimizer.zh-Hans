# 模拟您的历程{#simulation}

>[!CONTEXTUALHELP]
>id="ajo_journey_simulation"
>title="模拟您的历程"
>abstract="历程模拟功能允许您验证您的历程，并在激活之前查看其表现。它使用来自训练模型的数据来提供整个历程的数字，以了解历程在现实世界场景中的表现。"

历程模拟功能允许您验证您的历程，并在激活之前查看其表现。它使用来自训练模型的数据来提供整个历程的数字，以了解历程在现实世界场景中的表现。
