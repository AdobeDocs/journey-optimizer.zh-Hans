# 结束活动{#end-activity}

>[!BEGINSHADEBOX]

**在此页面上：**&#x200B;了解如何使用结束活动标记每个历程路径的结束并添加标签，使历程报告更易于阅读。

>[!ENDSHADEBOX]

>[!CONTEXTUALHELP]

>title="结束活动"
>abstract="您可以使用“结束活动”标记历程的每条路径的终点。 这并非强制性要求，但我们推荐使用以便能够清晰了解情况。 实际上，如果历程有多个结束活动，我们建议您为每个结束活动添加标签，以使报告更易于阅读。"

**[!UICONTROL End]**&#x200B;活动允许您标记历程每个路径的结尾。 这并非强制性要求，但我们推荐使用以便能够清晰了解情况。 实际上，如果历程有多个结束活动，我们建议您为每个结束活动添加标签，以使报告更易于阅读。 请参阅[此页](../reports/live-report.md)。
