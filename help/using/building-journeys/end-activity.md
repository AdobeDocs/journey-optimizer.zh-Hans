# 结束活动{#section_vqp_4ft_dgb}

![](../assets/do-not-localize/badge.png)

**[!UICONTROL End]**&#x200B;活动允许您标记旅程中每条路径的结束。 它不是强制性的，但建议使用它以实现视觉清晰度。 事实上，如果旅程有几个最终活动，我们建议您在每个结尾处添加一个标签，以便更轻松地阅读报告。 请参阅[此页](../reports/live-report.md)。
