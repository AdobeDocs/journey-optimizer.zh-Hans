# 创建优惠的关键步骤 {#key-steps}

创建选件的主要步骤如下：

1. **创建投放位置**.
版面是用于显示选件的容器。 例如，您可以创建一个仅以图像格式专用于选件的版面，该版面位于消息顶部。

1. **创建决** 策规则，以指定将根据哪些条件显示选件。

1. **创** 建要与选件关联的标记，以便轻松地将选件组织并搜索到库中。

1. **创建选件的内容和属性**。

1. **创建备用选件**，如果客户不符合任何选定选件的资格，这是最后显示的选件。
