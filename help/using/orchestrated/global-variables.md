# 在编排的营销活动中定义全局变量 {#define-global-variables}

**全局变量**&#x200B;是您在单个编排的营销活动中设置的名称 — 值对，可在每次运行时重复使用，因此您可以控制&#x200B;**[!UICONTROL 测试]**&#x200B;条件、规则生成器以及其他具有共享值的画布逻辑（例如，默认渠道或测试电子邮件），而无需将相同的值粘贴到每个活动中。

本页介绍如何定义全局变量。 这些变量可用后，有关如何在规则和&#x200B;**[!UICONTROL 测试]**&#x200B;条件中使用这些变量的详细信息，请参阅[在编排的营销活动中使用变量](variables-orchestrated-campaigns.md)。

要在编排的营销活动中添加或编辑全局变量，请执行以下步骤：

1. 打开“编排”营销活动。

1. 单击&#x200B;**[!UICONTROL 保存]**&#x200B;旁边的&#x200B;**...**&#x200B;图标，然后选择&#x200B;**[!UICONTROL 编辑变量]**。

   ![促销活动菜单中的“编辑变量”选项](assets/variables-list.png){zoomable="yes"}

1. 单击&#x200B;**[!UICONTROL 添加变量]**&#x200B;并设置变量的名称和值。 要编辑现有变量，请单击&#x200B;**...**&#x200B;按钮，然后选择&#x200B;**[!UICONTROL 编辑]**。

   ![用于添加或编辑变量的“变量”面板](assets/variables-edit.png){zoomable="yes"}

要了解如何在规则和&#x200B;**[!UICONTROL 测试]**&#x200B;条件中定义全局变量，请参阅[在编排的营销活动中使用变量](variables-orchestrated-campaigns.md#use)。
