# Journey Optimizer功能可用性 {#ajo-features-availability}

>[!BEGINSHADEBOX]

**在此页面上：**&#x200B;了解哪些[!DNL Adobe Journey Optimizer]功能可用，每个功能处于哪个生命周期阶段（“一般可用性”、“有限可用性”或“Beta”），它们适用于哪个基本选件，以及它们何时发货 — 这样您就可以回答&#x200B;*“我可以使用它吗？”* 无需仔细阅读发行说明。

>[!ENDSHADEBOX]

此页面整合了[!DNL Adobe Journey Optimizer]中的功能可用性，以便您能够确认在实施前范围设定期间可以使用的功能。 功能按功能区域分组。 在每个区域中，每个功能都列出了其当前的生命周期状态、其所适用的基本选件、其可用日期以及有关配置或区域限制的任何注释。

在&#x200B;*Available since*&#x200B;列中标记为&#x200B;**核心功能**&#x200B;的行是长期存在的基本功能，在2026年之前便已普遍可用。 已过期的行反映了2026年发布的更改。

>[!IMPORTANT]
>
>**为什么在我的环境中看不到功能？** **有限可用性**&#x200B;或&#x200B;**Beta**&#x200B;中的功能对所有人不可见 — 它们首先会转出到一组受限制的组织。 如果您阅读过的功能未出现在环境中，请检查其以下状态：如果它是&#x200B;**LA**&#x200B;或&#x200B;**Beta**，请联系您的Adobe代表以请求获取访问权限。 此处列出的功能并不表示在您的环境中启用了它。

>[!NOTE]
>
>**可用性与权利** 此页面跟踪&#x200B;*功能的生命周期和可用性*（功能是否已发运以及到期时间）。 许可证中是否包含功能&#x200B;**&#x200B;取决于您的基本选件和加载项 — 请参阅[包和功能](ajo-packages.md)。

## 生命周期状态的含义 {#status-definitions}

| 状态 | 它的含义 |
|--------|--------------|
| **GA** — 一般可用性 | 已发布到所有环境。 适用于其许可证包含该功能的任何组织。 |
| **LA** — 可用性有限 | 已发布到一组受限制的组织。 **联系您的Adobe代表**&#x200B;以请求获取访问权限。 |
| **Beta 版** | 用于评估的抢先体验版本。 在正式发布之前可能会发生更改。 可能需要选择加入。 |

## “应用于”如何映射到基本选件 {#applies-to}

**应用于**&#x200B;列引用了三个[!DNL Adobe Journey Optimizer]基本选件：

- **Journey Optimizer — 营销活动** — 批次、基于受众的编排
- **Journey Optimizer -历程** — 实时、事件驱动的编排
- **Journey Optimizer — 营销活动和历程** — 两者

标记为&#x200B;**所有基本选件**&#x200B;的渠道、内容和平台功能均可用，无论基本选件如何，但大多数选件仍需要相关的渠道或高级功能附加组件。 有关权利详细信息，请参阅[包和功能](ajo-packages.md)。
