# journeyStep 事件身份字段 {#sharing-identity-fields}

![](../assets/do-not-localize/badge.png)

此混音特定于journeyStepEvent:此事件与旅程相关，没有identityMap，用于描述用户档案身份（如果有）。

对于journeyStepEvent，我们还需要添加与标识相关的字段：

## profileID

用户档案标识符

类型：字符串

## profileNamespace

用户档案标识符命名空间

类型：字符串
