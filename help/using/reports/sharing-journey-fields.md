# 历程字段 {#sharing-journey-fields}

此字段组用在历程架构中（与journeyStepEvent相关）。 它包含以下字段：

## journeyID {#journeyid-field}

主历程的ID。

类型：字符串

## journeyVersionID {#journeyversionid-field}

历程版本的ID。 此id表示历程的标识。

类型：字符串

## name {#name-field}

历程的名称。

类型：字符串

## 描述 {#description-field}

历程的描述。

类型：字符串

## 版本 {#version-field}

版本，表示为 `major`.`minor`

类型：字符串
