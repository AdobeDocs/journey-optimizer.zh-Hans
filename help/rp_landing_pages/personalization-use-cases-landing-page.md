# 个性化用例{#section-overview}

本节提供了在Adobe Journey Optimizer中构建个性化消息的分步使用案例。 每个示例都基于真实情景，并展示了如何组合用户档案属性、上下文数据、辅助功能和优惠决策，以向受众提供有针对性的相关内容。

## 个性化用例

:::: landing-cards-container
:::
![icon](https://cdn.experienceleague.adobe.com/icons/circle-play.svg?lang=zh-Hans)

订单状态通知

逐步了解结合用户档案属性、实时上下文数据和Offer Decisioning的完整方案，向客户发送个性化的推送通知，以确认或更新其订单状态。

[阅读教程](../using/personalization/personalization-use-case.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/envelope.svg?lang=zh-Hans)

购物车放弃电子邮件

了解如何使用帮助程序功能动态地将特定于客户的详细信息和产品信息填充到购物车放弃电子邮件中，从而帮助您通过及时发送相关的消息来恢复销售额损失。

[阅读指南](../using/personalization/personalization-use-case-helper-functions.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/list-check.svg?lang=zh-Hans)

健康计划处方警报

使用条件逻辑和模板表达式，构建单个个性化电子邮件，其中列出当前处于“准备接收”或“已召回”状态的订阅者的所有处方。

[阅读示例](../using/personalization/perso-uc-plan-prescriptions.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/code-branch.svg?lang=zh-Hans)

Personalization配方

浏览现成的个性化模式库，其中涵盖日期、配置文件属性、条件逻辑等。 每个方法都包含一个工作语法片段，您可以将其直接复制到电子邮件、短信或推送内容中。

[浏览配方](../using/personalization/personalization-recipes.md)
:::
