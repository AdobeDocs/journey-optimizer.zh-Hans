# 个性化用例{#section-overview}

本节提供了在 Adobe Journey Optimizer 中构建个性化消息的分步用例。每个示例都基于实际场景，展示如何结合轮廓属性、上下文数据、辅助函数和优惠决策，向受众提供具有针对性的相关的内容。

## 个性化用例

:::: landing-cards-container
:::
![icon](https://cdn.experienceleague.adobe.com/icons/circle-play.svg)

订单状态通知

逐步讲解一个完整场景，该场景结合了轮廓属性、实时上下文数据和优惠决策，向客户发送个性化推送通知，以确认或更新其订单状态。

[阅读教程](../using/personalization/personalization-use-case.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/envelope.svg)

购物车放弃电子邮件

了解如何使用辅助函数，用客户特定的详细信息和产品信息动态填充购物车放弃邮件，通过相关且及时的消息帮助挽回流失的销售机会。

[阅读指南](../using/personalization/personalization-use-case-helper-functions.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/list-check.svg)

健康计划处方提醒

使用条件逻辑和模板表达式，构建一封个性化电子邮件，列出订阅者当前状态为“待取件”或“已召回”的所有处方。

[阅读示例](../using/personalization/perso-uc-plan-prescriptions.md)
:::

:::
![icon](https://cdn.experienceleague.adobe.com/icons/code-branch.svg)

个性化配方

浏览一个即用型个性化模式库，涵盖日期、轮廓属性、条件逻辑等。每个方法都包含一个可运行的句法片段，您可以直接将其复制到电子邮件、短信或推送内容中。

[浏览配方](../using/personalization/personalization-recipes.md)
:::
